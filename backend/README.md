# inmybeli backen

## Setup

From `backend/`:

```bash
python -m venv .venv
# Windows PowerShell:
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run

From `root`:

```bash
python -m backend.app
```

The server runs on `http://127.0.0.1:5001`.
