# Menuvo

Your social recipe book.

https://github.com/bobbybotbop/menuvo

# Description

Menuvo is a social recipe-sharing iOS app where users can create and share recipes, organize them into personal cookbooks, and follow what their friends are cooking and saving.

Key features:

- Create recipes with a title, description, ingredients, step-by-step instructions, cuisine, cook time, and a photo
- Organize recipes into named cookbooks, with a built-in "saved" cookbook for bookmarking others' recipes
- A social discovery feed that surfaces the newest recipes and highlights what your friends have saved
- Friend system with friend requests, pending requests, and user search
- Leave ratings and reviews on any recipe
- Profile pictures and recipe images stored in AWS S3

See backend/API_SPECIFICATIONS.md for full endpoint documentation and notes.
